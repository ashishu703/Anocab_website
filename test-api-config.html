📝 How to Use

Add this line to your HTML files:

<script src="/config.js"></script>

Then use API_BASE_URL in your code:

fetch(`${API_BASE_URL}/api/prices`)
  .then(res => res.json())
  .then(data => console.log(data));

To Change API URL:

  1. Edit backend/.env
  2. Change API_BASE_URL=your-url
  3. Restart backend server
  4. Reload this page
